# CRUD de Funcionários - Django

Projeto desenvolvido em Python utilizando o framework Django, conforme roteiro disponibilizado em aula.

## Tecnologias
- Python
- Django
- MySQL
- Bootstrap

## Funcionalidades
- Cadastro
- Listagem
- Visualização
- Edição
- Remoção

## Persistência
Os dados são armazenados em banco de dados MySQL.
